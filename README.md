# VisionX

## Descrierea proiectului:

**VisionX** este o aplicatie de tip website pentru gestionarea activitatii unui cinematograf. Proiectul va simula o platforma unde utilizatorii pot vizualiza intregul program al cinematografului.

Utilizatorii vor putea:
- Sa se conecteze utilizand un username 

- Sa acceseze programul filmelor disponibile in functie de:

    - ziua saptamanii
    - ora de ecranizare
    - sala si tipul(2D, 3D, IMAX)

- Vizualizarea detaliilor despre film (durata, gen)

- Rezervarea locurilor intr-o sala/cumpararea unui bilet pe care se va indica numele clientului si detaliile filmului.

- Vizualizarea preturilor si aplicarea reducerilor(elev/student).
